# 2025-haedal-bootcamp-submissions

# 해달 준회원 백준 과제 제출 레포

## 📁 제출 방식 (폴더 구조)
1. 이 레포를 **Fork**  
2. `problems/<BOJ_ID>/` 폴더 생성  
3. 그 안에 C 파일 10개(`problem-<번호>.c`) 업로드  

```text
problems/
└─ <BOJ_ID>/
   ├─ problem-17608.c
   ├─ problem-10988.c
   └─ … (총 10개)
```


## 📌 PR 작성 규칙
- **PR 제목**: `[준회원과제] BOJ_ID`  
- **PR 본문**:
  ```markdown
